# README

## 本アプリについて

本アプリは英単語を暗記することに特化した、暗記カードアプリです。<br/>
覚えたい単語を追加すると、暗記カードの裏に自動で意味や例文などが記入されます。<br/>
Next.js、Typescriptをメインに使用して作成しております。

## 使用した技術

- Next.js(Next 13.4.19)
- Typescript
- SCSS Modules
